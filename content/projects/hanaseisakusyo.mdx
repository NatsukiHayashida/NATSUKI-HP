## プロジェクト概要

「花製作所」は、妻が手作りするフラワーリース・スワッグを販売するオンラインショップです。このサイトは私が初めて制作したWebサイトであり、現在も継続的に運営・改善を行っています。

### 主な機能

- **商品カタログ**: 月2回更新される季節の花製品を魅力的に展示
- **レスポンシブデザイン**: スマートフォン・タブレット・PCすべてで最適な表示
- **SNS連携**: InstagramフィードとTwitter投稿の自動表示
- **問い合わせフォーム**: EmailJS統合によるスパム対策付き問い合わせ機能

## 技術スタック

### フロントエンド
- **Next.js 14** (App Router): サーバーサイドレンダリングと静的サイト生成
- **TypeScript**: 型安全性とコード品質の向上
- **Tailwind CSS**: 高速なスタイリングとデザインの一貫性

### デプロイ・運用
- **Vercel**: 自動デプロイとEdge Networkによる高速配信
- **Google Analytics**: ユーザー行動分析とコンバージョン追跡

## 実装のポイント

### 1. パフォーマンス最適化

商品画像が多いECサイトの特性上、画像最適化が最重要課題でした：

```typescript
// next/imageを活用した最適化
<Image
  src="/products/wreath-spring.jpg"
  alt="春のフラワーリース"
  width={600}
  height={600}
  priority={index < 2} // Above the fold画像のみpreload
  quality={85}
  placeholder="blur"
  blurDataURL="/placeholder.jpg"
/>
```

結果として、**Lighthouse Performance Score 95+**を達成。

### 2. SEO対策

構造化データと適切なメタタグにより、Google検索結果での視認性を向上：

```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "春のフラワーリース",
  "image": "https://www.hanaseisakusyo.com/products/wreath-spring.jpg",
  "description": "手作りの春らしいフラワーリース",
  "offers": {
    "@type": "Offer",
    "price": "5000",
    "priceCurrency": "JPY"

```

### 3. モバイルファーストデザイン

ユーザーの70%以上がモバイルからのアクセスであることを考慮し、タッチ操作に最適化：

- タップ領域は最低44×44px
- スワイプによる画像ギャラリー
- モバイルメニューの改善
